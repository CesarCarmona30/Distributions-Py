## Theory


### Probability Mass Function (PMF)

$$p(x; k) =
    \begin{cases}
     \frac{\euler^{\lambda} \lambda^x}{x!}, & \quad x = 0, 1, 2, \dots; \lambda > 0\\
     0, & \quad otherwise
    \end{cases}
$$
